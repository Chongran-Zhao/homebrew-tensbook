# Homebrew Tensbook

Homebrew tap for the Tensbook macOS app.

## Install

```sh
brew install --cask Chongran-Zhao/tensbook/tensbook
```

Homebrew requires the full `owner/tap/cask` form for untapped casks. The shorter
`brew install --cask Chongran-Zhao/tensbook` only names the tap, not the app.

## Update

```sh
brew update
brew upgrade --cask tensbook
```

## Uninstall

```sh
brew uninstall --cask tensbook
```
